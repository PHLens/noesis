# Examples

## Research Proposal

Input:

```text
Goal: Before enrollment in July/August, turn stateful LLM inference into an advisor-ready proposal using available Cambricon MLU resources.
Horizon: 8 weeks.
Constraints: public models/data only; B-level profiler is usable; A-level profiler needs confirmation.
Artifacts: LoreForge proposal page, stateful-llm repo, Slock task thread.
```

Weekly outcome:

```markdown
## Focus

- [plan/stateful-llm] Establish a public-data Phase 1 measurement base on MLU.

## Outcomes

- [ ] Define the first workload matrix with shapes, metrics, and data boundary.
- [ ] Run or delegate one smoke measurement path.
- [ ] Record evidence and gaps in the weekly note and linked repo docs.

## Work Blocks

- [ ] Inspect current proposal/wiki pages and list Phase 1 assumptions.
- [ ] Convert assumptions into workload matrix cases.
- [ ] Ask for missing profiler/export constraints if needed.
- [ ] Update repo docs or issue notes with public-data-only boundaries.

## Daily Projection

- Mon: [plan/stateful-llm] Read proposal + repo scaffold; list Phase 1 assumptions.
- Tue: [plan/stateful-llm] Draft workload matrix and metric schema changes.
- Wed: [plan/stateful-llm] Validate with MLU machine owner; revise next actions.
```

Daily TODOs:

```markdown
## TODO

- [ ] [plan/stateful-llm] Read the proposal page and write the Phase 1 assumption list into this week's note.
- [ ] [plan/stateful-llm] Check `stateful-llm` repo docs for current workload matrix gaps.
```

## Career Or Study Goal

Input:

```text
Goal: Prepare for a systems internship search.
Horizon: 6 weeks.
Constraint: 6 hours per week outside full-time work.
```

Weekly outcome:

```markdown
## Focus

- [plan/internship] Produce one visible systems artifact and one application-ready narrative.

## Outcomes

- [ ] Publish a concise project writeup or demo note.
- [ ] Draft a 150-word pitch connecting the project to target roles.

## Work Blocks

- [ ] Pick one project artifact with strongest signal.
- [ ] Extract measurable results and constraints.
- [ ] Draft the pitch.
- [ ] Review against two target job descriptions.
```
