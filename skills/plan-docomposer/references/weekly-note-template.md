# Weekly Note Template

Use this structure for Obsidian weekly notes created or updated by `plan-docomposer`.

```markdown
---
date: "{{date:gggg-[W]ww}}"
type: weekly
tags:
  - weekly
---

# {{title}}

## Focus

-

## Active Plans

-

## Outcomes

- [ ]

## Work Blocks

- [ ]

## Daily Projection

- Mon:
- Tue:
- Wed:
- Thu:
- Fri:
- Sat:
- Sun:

## Risks / Blockers

-

## Decisions

-

## Review

- Done:
- Learned:
- Carry forward:
```

## Section Semantics

- `Focus`: one short sentence or 1-3 bullets describing the week-level theme.
- `Active Plans`: plan IDs or project names, e.g. `[plan/stateful-llm]`.
- `Outcomes`: deliverables that make the week count as successful.
- `Work Blocks`: chunks of work that can be moved across days.
- `Daily Projection`: light allocation for the next few days; do not overfill the whole week if facts will change.
- `Risks / Blockers`: constraints, dependencies, and likely failure modes.
- `Decisions`: choices made this week that should be visible during review.
- `Review`: filled at the end of the week.
